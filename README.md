# VisualHTML

VisualHTML es un visor HTML simple y rápido para probar código en tiempo real.

## Qué hace

- Editor arriba para insertar tu HTML.
- Vista previa abajo que se actualiza al instante.
- Proyecto de una sola página (`index.html`) , disponible en 

## Estructura

- `index.html`: aplicación principal.

## Uso local

1. Abre `index.html` en tu navegador.

2. Escribe o pega tu HTML en el editor superior.

3. Mira el resultado en el preview inferior.

Disponible en [GitHub Pages](https://LordAlastor78.github.io/VisualHTML/) para acceso instantáneo.
